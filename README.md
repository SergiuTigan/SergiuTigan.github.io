# InvictiDemo

This project was for the purpose of showcasing my skills for a job interview.

This project offers the posibility to display a list of users into a datatable
created using TailwindCSS. Upon clicking on a row, a card appears showing all the 
information a user has.
Technologies
##Prerequisits
- [NodeJs](https://nodejs.org/en/)
- [Node Package Manager](https://docs.npmjs.com/)
- Angular CLI( npm install -g @angular/cli )

## Install Steps
- Run `npm install`
- Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`.
